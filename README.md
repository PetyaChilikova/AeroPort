# AeroPort
AeroPort е обектно-ориентирано конзолно приложение на C++ за управление на летищни процеси. Системата разграничава три нива на достъп, позволяващи планиране на полети (диспечери), покупка и модификация на билети (пътници) и централизиран мениджмънт на летищната инфраструктура и финанси (администратори).
